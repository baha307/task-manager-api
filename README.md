# task-manager-api

A simple full-stack task management application built using Django and Django REST Framework.  
The project demonstrates backend development concepts such as user authentication, REST APIs, database management, and CRUD operations.

---

## Features

### User Management
- User registration
- User login authentication
- JWT-based authentication using Django REST Framework Simple JWT

### Task Management
- Create tasks
- View tasks
- Update tasks
- Delete tasks
- Assign tasks to users
- View user-specific tasks

### API Features
- RESTful API endpoints
- JSON-based responses
- Filtering support
- Pagination support

### Database
- SQLite database for development
- Django ORM for database operations
